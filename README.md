# SIM800 GPS Module

This project presents the **PCB design and implementation** of a SIM800 GSM module integrated with a GPS module using **Altium Designer**.  
It includes the complete Altium project files, schematics, PCB layout, and photos of the assembled module.  
The design enables GSM communication and GPS positioning for embedded applications and IoT devices.

### Tools Used
- Altium Designer
- SIM800 GSM Module
- GPS Module

### Project Contents
- Complete Altium project folder (.PrjPCB, .SchDoc, .PcbDoc, etc.)
- PCB layout files
- Photos of the assembled PCB
